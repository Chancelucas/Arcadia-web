- Installer Homebrew pour mac : 
    1 - commande dans le terminal : /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

- Installation de php (pour mac avec Homebrew): 
    1 - commande dans le terminal : brew install php

- Installation de composer (pour mac avec Homebrew):
    1 - commande dans le terminal : brew install composer

- Installer cloudinary : 
    1 - https://github.com/cloudinary/cloudinary_php?tab=readme-ov-file
    2 - commande dans le terminal : composer require "cloudinary/cloudinary_php"
    3 - Dans le dossier config du projet crée un fichier : config.php avec les info fourni par cloudinary :
        3.1 - a inscrire dans ce fichier : 
            use Cloudinary\Configuration\Configuration;

            Configuration::instance([
                'cloud' => [
                    'cloud_name' => 'VOTRE_CLOUD_NAME', 
                    'api_key' => 'VOTRE_API_KEY', 
                    'api_secret' => 'VOTRE_API_SECRET'
                ],
                'url' => [
                    'secure' => truegit 
            ]
            ]);
